# Frontend

React + Vite + TypeScript app for the AI resume portfolio.

## Install

```bash
npm install
```

Verification: `node_modules/` exists and `package-lock.json` is up to date.

## Build

```bash
npm run build
```

Verification: Vite writes production assets into `dist/`.

## Run

```bash
npm run dev
```

Verification: open the Vite local URL and confirm the Health card can reach the backend when `http://127.0.0.1:8000` is running.
